## 🔍 关键技术点

### 核心技术
- **RAG架构**: Retrieval-Augmented Generation
- **向量数据库**: ChromaDB + HNSW索引
- **Embedding方案**: 4种可选 (智能降级)
- **LLM集成**: DeepSeek + OpenAI (可选)
- **Web框架**: Gradio 5.0

### 技术亮点
- ✨ **零成本运行**: 完全免费的embedding方案
- ✨ **智能降级**: API不可用时自动切换
- ✨ **本地隐私**: 所有数据本地存储
- ✨ **灵活配置**: 从零配置到高级定制
- ✨ **高可用性**: 多层错误处理和降级策略

## ✨ 快速开始

**5分钟快速体验**:
```bash
# 1. 克隆项目
git clone <repo-url>
cd personal_knowledge_agent

# 2. 安装依赖
pip install -r requirements.txt
pip install sentence-transformers  # 推荐

# 3. 启动系统
python main.py
```

浏览器自动打开 `http://127.0.0.1:8888`，开始使用！
